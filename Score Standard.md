# US Liquidity Score Standard (11 Indicators)

## 1. Overview
- 목적: 미국 주식시장용 **유동성 신호등**(완화·중립·긴축)을 11개 지표를 모두 반영한 점수 체계로 판단하기 위한 기준.
- 범위: FRED 및 주요 시장지표를 활용해 **11개 지표 종합 점수(0–100점)** 를 산출하고, 이를 완화·중립·긴축으로 분류.
- 대상 지표(11개):
  1. RRP 잔고
  2. 은행 준비금
  3. TGA 잔고
  4. SOFR
  5. DXY (달러 인덱스)
  6. 장단기 금리차 (예: 10Y–2Y)
  7. MMF 자산
  8. M2 통화량
  9. 하이일드 스프레드
  10. VIX 공포지수
  11. Fear & Greed Index

---

## 2. Score Architecture

### 2.1 총점 구조
- 총점: **0–100점**.
- 11개 개별 지표를 모두 사용.
- 각 지표는 **0–10점**(5단계: 0 / 2.5 / 5 / 7.5 / 10)으로 평가.
- 가중 평균으로 총점을 구함.

### 2.2 지표별 가중치

| 구분        | 지표                   | 가중치(%) | 방향성 해석                          |
| ----------- | ---------------------- | --------: | ------------------------------------ |
| 기초 유동성 | RRP 잔고               |         8 | 감소 = 완화, 증가 = 긴축             |
|             | 은행 준비금            |        10 | 증가 = 완화, 감소 = 긴축             |
|             | TGA 잔고               |         8 | 감소 = 완화, 증가 = 긴축             |
|             | M2 통화량              |         8 | 증가 = 완화, 감소 = 긴축             |
|             | MMF 자산               |         6 | 감소·정체 = 완화, 급증 = 긴축        |
| 금리·달러   | SOFR                   |         6 | 하락·안정 = 완화, 급등 = 긴축        |
|             | 장단기 금리차 (10Y–2Y) |         8 | 플러스 확대 = 완화, 역전 심화 = 긴축 |

## 3. Indicator-Level Scoring Rules (0–10점)

각 지표는 **최근 3개월(13주)** 기준으로 방향성과 수준을 평가한다.

### 3.1 공통 점수 스케일

- 10점: 강한 완화 (유동성 공급·위험선호에 매우 우호적)
- 7.5점: 온건 완화
- 5점: 중립
- 2.5점: 온건 긴축
- 0점: 강한 긴축 (유동성 흡수·리스크오프 강함)

### 3.2 RRP 잔고 (8%)

- 지표: Fed ON RRP 잔고 (예: RRPONTSYD 등).
- 해석: **감소할수록 완화**, 증가할수록 긴축.[web:14][web:71]
- 기준: 최근 13주 변화율 기준.
  - 10점: 변화율 **-15% 이하**
  - 7.5점: -15% ~ -5%
  - 5점: -5% ~ +5%
  - 2.5점: +5% ~ +15%
  - 0점: +15% 이상

### 3.3 은행 준비금 (10%)

- 지표: Reserves of Depository Institutions (TOTRESNS 등).[web:19][web:78]
- 해석: **증가할수록 완화**, 감소할수록 긴축.
- 기준: 최근 13주 변화율.
  - 10점: **+5% 이상**
  - 7.5점: +2% ~ +5%
  - 5점: -2% ~ +2%
  - 2.5점: -5% ~ -2%
  - 0점: -5% 이하

### 3.4 TGA 잔고 (8%)

- 지표: Treasury General Account 잔고.[web:21][web:70]
- 해석: **감소할수록 완화**, 증가할수록 긴축.
- 기준: 최근 13주 변화율.
  - 10점: **-15% 이하**
  - 7.5점: -15% ~ -5%
  - 5점: -5% ~ +5%
  - 2.5점: +5% ~ +15%
  - 0점: +15% 이상

### 3.5 SOFR (6%)

- 지표: Secured Overnight Financing Rate.[web:39][web:66][web:72]
- 해석: 단기 달러 자금 조달비용. **안정·하락 = 완화**, 급등 = 긴축.
- 기준: 최근 4주 평균과 13주 평균 차이 (bp 기준).
  - 10점: 4주 평균이 13주 평균 대비 **-10bp 이하**
  - 7.5점: -10bp ~ -3bp
  - 5점: -3bp ~ +3bp
  - 2.5점: +3bp ~ +10bp
  - 0점: +10bp 이상

### 3.6 DXY (6%)

- 지표: 달러 인덱스 DXY.[web:40][web:81]
- 해석: **달러 약세 = 글로벌 유동성 완화**, 강세 = 긴축.
- 기준: 최근 13주 변화율.
  - 10점: **-3% 이하**
  - 7.5점: -3% ~ -1%
  - 5점: -1% ~ +1%
  - 2.5점: +1% ~ +3%
  - 0점: +3% 이상

### 3.7 장단기 금리차 (8%)

- 지표: 10년–2년(또는 10년–3개월) 국채 수익률 차이.[web:40]
- 해석: **플러스·정상화 방향 = 완화/성장 기대**, 역전 심화 = 긴축·경기우려.
- 기준: 현재 스프레드 레벨.
  - 10점: **+0.75%p 이상**
  - 7.5점: +0.25%p ~ +0.75%p
  - 5점: 0%p ~ +0.25%p
  - 2.5점: -0.50%p ~ 0%p
  - 0점: -0.50%p 이하

### 3.8 MMF 자산 (6%)

- 지표: 미국 MMF(머니마켓펀드) 전체 자산.[web:63][web:61]
- 해석: **급증 = 안전자산 선호·리스크오프 = 긴축적**, 감소·정체 = 완화적.
- 기준: 최근 13주 변화율.
  - 10점: **-5% 이하**
  - 7.5점: -5% ~ -1%
  - 5점: -1% ~ +1%
  - 2.5점: +1% ~ +5%
  - 0점: +5% 이상

### 3.9 M2 통화량 (8%)

- 지표: M2 Money Stock (M2SL/WM2NS).[web:48][web:51]
- 해석: **증가 = 완화**, 감소 = 긴축.
- 기준: 최근 13주 변화율.
  - 10점: **+2.5% 이상**
  - 7.5점: +1.0% ~ +2.5%
  - 5점: -1.0% ~ +1.0%
  - 2.5점: -2.5% ~ -1.0%
  - 0점: -2.5% 이하

### 3.10 하이일드 스프레드 (10%)

- 지표: ICE BofA US High Yield Index Option-Adjusted Spread 등 (bps).[web:27][web:49][web:34]
- 해석: **스프레드 축소 = 위험선호·완화**, 확대 = 위험회피·긴축.
- 기준: 현재 스프레드 레벨.
  - 10점: **300bp 미만**
  - 7.5점: 300–399bp
  - 5점: 400–499bp
  - 2.5점: 500–599bp
  - 0점: 600bp 이상

### 3.11 VIX 공포지수 (8%)

- 지표: CBOE VIX Index.[web:58][web:60][web:62][web:65]
- 해석: **낮음 = 안정·완화**, 급등 = 공포·긴축.
- 기준: 현재 레벨.
  - 10점: **VIX < 20**
  - 7.5점: 20–24.99
  - 5점: 25–29.99
  - 2.5점: 30–34.99
  - 0점: 35 이상

### 3.12 Fear & Greed Index (12%)

- 지표: CNN Fear & Greed Index (0–100).[web:32][web:35][web:55][web:56]
- 해석: **높음(탐욕) = 위험선호·완화**, 낮음(공포) = 긴축.
- 기준: 현재 지수.
  - 10점: **70–100** (강한 탐욕)
  - 7.5점: 55–69
  - 5점: 40–54
  - 2.5점: 25–39
  - 0점: 0–24 (공포/극단적 공포)

## 4. Total Score Calculation

각 지표 점수 \(S_i\) (0–10)을 가중치 \(w_i\) (%)와 곱해 합산한다.[web:40]

\[
\text{Total Score (0–100)} = \sum_{i=1}^{11} S_i \times \frac{w_i}{10}
\]

- 구현 팁: `Total = 10 × Σ(S_i × weight_i)` (weight_i는 0–1 사이 소수).

## 5. Regime Classification (완화·중립·긴축)

### 5.1 기본 총점 구간

- **완화 (Green)**: 총점 **70–100점**
- **중립 (Yellow)**: 총점 **40–69점**
- **긴축 (Red)**: 총점 **0–39점**

### 5.2 확인조건 (필터링 규칙)

#### 완화 확정 조건

1. 총점 ≥ 70점  
2. (은행 준비금 + M2) 평균 점수 ≥ 7.5점  
3. (RRP, TGA 중 최소 1개) 점수 ≥ 7.5점  
4. VIX 점수 ≥ 5점, Fear & Greed 점수 ≥ 5점  

#### 긴축 우선 조건

1. 총점 ≤ 39점  
2. (은행 준비금 + M2) 평균 점수 < 5점  
3. RRP, TGA, 하이일드 스프레드, VIX, DXY 중 **3개 이상**이 2.5점 이하  
4. 4주간 총점 변화가 **-10점 이상** 급락  

#### 중립 판정

- 기본적으로 총점 40–69점 구간은 중립.
- 총점이 70 이상이라도, VIX·하이일드 스프레드·Fear & Greed가 동시에 긴축 신호(2.5점 이하)를 보이면 **중립(완화 경계)** 로 처리.
- 총점이 39 이하가 아니더라도, 핵심 유동성 축(RRP, 준비금, TGA, M2)이 상충될 때는 중립 유지.

## 6. Operational Rules

- 업데이트 주기: **주간** (일간 지표는 주간 평균 사용).[web:39][web:48][web:63][web:70]
- 경계값 근처에서는 **2주 연속** 같은 구간 유지 시 레짐 변경.
- 정책 이벤트 직후 1주일은 참고용으로만 보고, 2–3주 추세 확인 후 판단.[web:78][web:87]
